# @rte/cli

Onboarding and diagnostics for rte in Vite + React + Tailwind projects.

```bash
bunx @rte/cli init --yes
bun run dev
```

**Commands:** `init` · `doctor` · `scan` · `stats`

See [rte user guide](https://github.com/ehah/Rte/blob/main/docs/rteUser.md) and [1.0 release notes](https://github.com/ehah/Rte/blob/main/docs/rte_v1.0.md).
