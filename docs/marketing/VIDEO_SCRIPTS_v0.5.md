# v0.5.0 demo video scripts (§23 minimum)

Two short recordings for launch. Capture with **Developer Details OFF**, 1440×900.

## Video 1 — Card spacing (§23)

**Prompt:** “Make this card smaller.”

**Flow:**

1. `pnpm dev:tailadmin` → open dashboard → Edit on.
2. Click **Orders Card** → **Card Style**.
3. Padding → **Compact** or **Medium** → **Preview Changes** → **Apply to Code**.
4. Show HMR update on the card.

**Voiceover:** Nuvio edits real React source — no re-prompting your AI for spacing tweaks.

## Video 2 — Table title (§23)

**Prompt:** “Rename Recent Orders.”

**Flow:**

1. Same app → **Recent Orders** table menu → **Table Title**.
2. Change title → Preview → Apply.
3. Optional: column header rename via **Column Headers**.

**Voiceover:** Tables, headers, and row text — same Preview → Apply flow.
